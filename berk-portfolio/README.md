# Berk Göktaş Portfolio

Static portfolio site built with semantic HTML, custom CSS, and vanilla JavaScript.

## Files

- `index.html` — page structure and copy
- `styles.css` — responsive visual system, dark/light themes
- `script.js` — mobile navigation, theme persistence, and safe profile-guide widget
- `assets/Berk_Goktas_CV.pdf` — downloadable CV

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

This folder can be deployed as a static site on GitHub Pages, Netlify, Cloudflare Pages, or Vercel.

Before publishing, replace any URL placeholders if the final public domain differs from your actual domain.
